Developer First

Ship Agents In Minutes

Simple, intuitive API that gets out of your way. Define agents, wire up tools, and deploy with a few lines of code.

  • ✓ Type-safe SDKs for Python & TypeScript
  • ✓ Built-in retry, fallback & rate limiting
  • ✓ Streaming responses out of the box
  • ✓ OpenTelemetry compatible tracing
orchestrate.py
from nexusai import Agent, Orchestrator

# Define specialized agents
researcher = Agent(
    "research-agent",
    model="gpt-4o",
    tools=[web_search, arxiv, wiki]
)

coder = Agent(
    "code-agent",
    model="claude-sonnet",
    tools=[code_exec, git, tests]
)

# Create orchestration pipeline
pipeline = Orchestrator([researcher, coder])
result = await pipeline.run(
    "Build a REST API for user auth"
)
